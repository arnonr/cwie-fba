# CLAUDE.md — CWIE-FBA Project

## Project Overview

**CWIE_KMUTNB** — Cooperative Work-Integrated Education management system for King Mongkut's University of Technology North Bangkok (KMUTNB). Manages student internship/co-op workflows including company selection, form submissions, teacher visits, and document handling.

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Laravel 10.8, PHP 8.1 |
| Frontend | Vue 3.2 + Vite 4.3, Vuetify 3.2 |
| State | Pinia 2 |
| Routing | Vue Router 4 (file-based via Vite Pages plugin) |
| Auth | Laravel Passport 11 + Sanctum 3 |
| Database | MySQL 8.0 |
| HTTP Client | Axios |
| Authorization | CASL (attribute-based) |
| Testing | PHPUnit 10 |
| Containerization | Docker + Docker Compose |

## Frontend Conventions

- **File-based routing:** Pages in `resources/js/pages/` auto-generate routes. No manual route registration needed for new pages.
- **Auto-imports:** Vue, Pinia, Vue Router composables and common utilities are auto-imported — no need to explicitly import them.
- **Component auto-registration:** Components in `resources/js/components/` are auto-registered globally.
- **Layouts:** Use the `<route>` block or layout meta in page components to specify layout.
- **Authorization:** CASL abilities configured per user role. Check `store/` for ability definitions.

---

## Backend Conventions

- **RESTful API only:** Frontend is a SPA; backend serves JSON API under `/api/*` with `auth:api` middleware.
- **Controller per resource:** One controller per entity (e.g., `StudentController`, `FormController`, `VisitController`).
- **Eloquent ORM:** Use Eloquent relationships, not raw queries.
- **Mass assignment:** All models use `$fillable` arrays.
- **Timestamps:** Models track `created_at`, `updated_at`, `created_by`, `updated_by`.

## Environment Setup

Copy `.env.example` to `.env` and configure:

```env
APP_URL=http://localhost:8117
DB_CONNECTION=mysql
DB_HOST=mysql          # or 127.0.0.1 outside Docker
DB_PORT=3306
DB_DATABASE=cwie_fba
DB_USERNAME=cwie_fba
DB_PASSWORD=2023@CWIEFBA
```

After setup:
```bash
composer install
npm install
php artisan key:generate
php artisan migrate
php artisan passport:install
```

## Key Domain Concepts

- **Student** — applies for co-op placement, submits forms, is visited by teachers
- **Form** — internship application form with status/approval workflow and reject logs
- **Visit** — teacher site visit to a student's workplace; has its own reject log
- **Company** — workplace where student does co-op
- **Teacher / MajorHead / Chairman** — different staff roles with different access levels
- **Semester** — academic period; scopes most data
- **Faculty / Department / Major** — academic hierarchy

## PDF & Document Features

Multiple PDF libraries are used (jspdf, pdf-lib, html2pdf.js, vue3-html2pdf). PDF generation is a core feature for official co-op documents and receipts.

## Language

The application targets Thai university users. UI text uses Vue i18n. Thai-specific utilities include `bahttext` (number-to-Thai-text conversion).
